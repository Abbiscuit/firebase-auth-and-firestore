## Firebase Auth & Firestore with React

Firebase Auth と Firestore を使ったユーザー認証

`firebase/firebase.utils.js`内に各種設定あり

### Firebase Auth

- Email & Password による認証
- `createUserWithEmailAndPassword`
- `signInWithEmailAndPassword`

- Google アカウントによる認証
- `signInWithGoogle`
